# Sobre mim
Olá! Eu iniciei na computação em 2019 quando eu tinha 14 anos de idade, a primeira linguagem de programação que eu tive experiência foi uma linguagem proprietaria de um jogo, sua sintaxe é baseada na BASIC, nela era necessário conhecimentos de baixo nível e engenharia reversa para criar novos comportamentos para o jogo. Posteriormente estudei por conta própria as linguagens C/C++ e utilizo para projetos pessoais, também tenho projetos que usam C# para criação de games na Unity. Em 2023 eu completei um BootCamp de programação Java Backend, no mesmo ano tive experiência com JavaScript. Um dos meus hobbies é estudar hardwares antigos e criar programas para esses hardwares, tenho projetos sobre o tema no meu GitHub. Eu sou uma pessoa do backend em que adora implementar novas funcionalidades e desenvolver sistema, estou em constante aprendizado pessoalmente e profissionalmente!
